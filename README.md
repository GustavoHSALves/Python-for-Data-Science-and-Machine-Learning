# 🐍 Python for Data Science and Machine Learning

This repository contains my notes, exercises, and practice notebooks from a Python for Data Science and Machine Learning course.

The goal of this repo is to consolidate fundamental concepts in Python, NumPy, and Pandas, as well as to build a solid foundation for data analysis and future machine learning applications.

---

## 📂 Repository Structure

The notebooks are organized by topic and progression:

### 🔹 Python Basics
- 01-Python Crash Course.ipynb  
- 02-Python Crash Course Exercises.ipynb  
- 03-Python Crash Course Exercises - Solutions.ipynb  

### 🔹 NumPy
- 01-NumPy Arrays.ipynb  
- 02-Numpy Indexing and Selection.ipynb  
- 03-Numpy Operations.ipynb  
- 04-Numpy Exercises.ipynb  

### 🔹 Pandas
- 01-Introduction to Pandas.ipynb  
- 02-Series.ipynb  
- 03-DataFrames.ipynb  
- 04-Missing Data.ipynb  
- 05-Groupby.ipynb  

### 🔹 Practice Exercises
- SF Salaries Exercise  
- Ecommerce Purchases Exercise  

Each exercise also includes solution notebooks for reference.
